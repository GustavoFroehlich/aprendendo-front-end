Flex-Direction

Ordem de alinhamento dos seus elementos filho.

Existe 4 propriedades.

Row(Padrão):

-A ordem de alinhamento fica no formato de linha. Um elemento ao lado do outro.

3
4

Column:

-A ordem de alinhamento fica no formato de coluna. Um elemento em cima do outro

3
4

Row-reverse:

-A ordem de alinhamento fica no formato de linha, mas ela inverte a ordem de trás pra frente dos elementos. Um elemento ao lado do outro na ordem contrária e no outro lado da página.

Column-reverse:

-A ordem de alinhamento fica no formato de coluna, mas ela inverte a ordem de baixo pra cima dos elementos. Um elemento ao em cima do outro na ordem contrária.

Flex-Wrap

Diz para o elemento pai o que fazer quando o espaço não é suficiente para alinhar os elementos na nossa ordernação(Flex-direction).

Existe 3 propriedades.

Nowrap:

-Ele não quebra a linha ou coluna, apenas achata tudo lá dentro.

Row:

(O quadrado grande contém: 200px de altura e 150px largura);

(Cada quadradinho contém: 50px de altura e largura, somando dá 200px de altura);

O flexbox achatou os quadradinhos na parte da largura, para que continuassem cabendo dentro do quadrado grande!

1
2
3
4

Column:

(O quadrado grande contém: 150px de altura e 150px de largura);

(Cada quadradinho contém: 50px de altura e largura, somando dá 200px de largura);

O flexbox achatou os quadradinhos na parte da altura, para que continuassem cabendo dentro do quadrado grande!

1
2
3
4

Wrap:

-Quando não couber mais na mesma linha ou coluna, eles vão quebrar e criar uma linha ou coluna nova sem achatar nada.

Row:

(O quadrado grande contém: 200px de altura e 150px de largura);

(Cada quadradinho contém: 50px de altura e largura, somando dá 200px de largura);

O Flexbox criou uma nova linha para não achatar os quadradinhos!

1
2
3
4

Column:

(O quadrado grande contém: 150px de altura e 150px de largura);

(Cada quadradinho contém: 50px de altura e largura, somando dá 200px de largura);

O Flexbox criou uma nova coluna para não achatar os quadradinhos!

1
2
3
4

Wrap-reverse:

-Quando não couber mais na mesma linha ou coluna, eles vão quebrar e criar uma linha ou coluna nova sem achatar nada, mas eles irão inverter a ordem dos elementos.

Row:

(O quadrado grande contém: 200px de altura e 150px de largura);

(Cada quadradinho contém: 50px de altura e largura, somando dá 200px de largura);

O Flexbox criou uma nova linha para não achatar os quadradinhos! E inverteu a ordem dos elementos.

Column:

(O quadrado grande contém: 150px de altura e 150px de largura);

(Cada quadradinho contém: 50px de altura e largura, somando dá 200px de largura);

O Flexbox criou uma nova coluna para não achatar os quadradinhos! E inverteu a ordem dos elementos.

Flex-Flow

O flex-flow, refere-se à uma Shorthand para flex-direction e flex-wrap, fazendo com que você possa declarar as duas tags mais facilmente.

flex-flow: (Flex-direction) (Flex-wrap); Exemplo: flex-flow: row wrap
